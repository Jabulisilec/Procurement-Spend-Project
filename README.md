# Procurement Spend Project

## Overview
This project aims to analyze procurement spending patterns and identify key insights that can help organizations optimize their purchasing strategies.

## Project Structure
```
Procurement-Spend-Project/
├── data/
│   └── datasets/
├── analysis/
│   └── analysis.ipynb
├── reports/
└── README.md
```

## Dataset Description
The dataset consists of various procurement transactions, including:
- Transaction ID
- Supplier
- Category
- Amount Spent
- Date of Purchase

## Analysis File Details
The `analysis/analysis.ipynb` file contains:
- Data cleaning and preprocessing scripts
- Exploratory Data Analysis (EDA)
- Visualizations to identify spending patterns

## Key Insights
1. The top 5 suppliers account for 70% of the total spend.
2. Seasonal trends in procurement spending are noticeable, with peaks during mid-year.
3. Certain categories show a disproportionately high spend compared to others.

## Getting Started Guide
1. Clone the repository:
   ```
   git clone https://github.com/Jabulisilec/Procurement-Spend-Project.git
   ```
2. Navigate to the project directory:
   ```
   cd Procurement-Spend-Project
   ```
3. Install required packages:
   ```
   pip install -r requirements.txt
   ```
4. Launch the analysis notebook:
   ```
   jupyter notebook analysis/analysis.ipynb
   ```

## Recommendations
- Regularly review spending patterns to make timely adjustments in procurement strategies.
- Explore additional data sources to enrich the dataset for more robust analyses.
